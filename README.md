# 🚀 Portfólio Tech | Vitrine Digital High Ticket

Uma landing page premium desenvolvida com foco em **conversão, autoridade e inteligência de dados**. Este projeto vai além de uma simples apresentação visual; é uma ferramenta estratégica construída para gerar resultados e otimizar a captação de leads.

## 🔗 Links Oficiais
- **Deploy ao vivo:** [Acessar o Portfólio](https://portfolio-opal-omega-7kjyjot4r6.vercel.app)

## 🛠️ Tecnologias e Recursos Aplicados

- **UI/UX Avançado:** Design em *Dark Mode* estruturado para retenção de atenção e conforto visual.
- **Glassmorphism:** Efeito visual moderno e elegante estilo "vidro fosco" utilizando CSS3 puro.
- **Vanilla Tilt (3D):** Interatividade imersiva nos elementos da interface através de manipulação do DOM com JavaScript.
- **Integração WhatsApp API:** Botões de *Call to Action* (CTA) focados na redução de atrito para o fechamento rápido de propostas.
- **Inteligência de Dados:** Rastreamento de conversões implementado via **Meta Pixel** direto no core da aplicação, preparando a estrutura para campanhas de tráfego pago.
